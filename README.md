# WLSP-norms (version 1.0) (2025/11/10)

## Description
『分類語彙表』増補改訂版データベースに対して、クラウドソーシングにより具体性・抽象性・古さ・新しさ・多義性を付与したもの

## Features

### abstract.txt, concrete.txt, innovative.txt, obsolete.txt, polysemous.txt

1行目がヘッダ。

- 1列目：WID 単語ID
- 2列目：WORD 単語
- 3列目：LABEL 分類語彙表 (`独自のID`-`類`-`部門`-`中項目`-`分類項目`-`分類番号`)
- 4列目：FACTOR 評定種別 (ABSTRACT, CONCRETE, INNOVATIVE, OBSOLETE, POLYSEMOUS)
- 5列目：VALUE 正規化推定値
